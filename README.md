# Titanic ML Learning Tool

Neeche are the instructions for running the application

## Running the Application

### 1. Install Streamlit

```bash
pip install streamlit
```

### 2. Change directory

```bash
cd project_ui
```

### 3. Run the application

```bash
streamlit run app.py
```

The application will open in your default web browser. If it doesn't open automatically, navigate to the URL shown in your terminal (typically `http://localhost:8501`).
